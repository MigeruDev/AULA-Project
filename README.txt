Este proyecto cuenta con una base de datos implementada
en SQLite. La misma que se adjunta en esta carpeta.

Si desea utilizar la base de datos (recomendado), deberá
acceder al código fuente de la clase ConexionSQLite,
la misma que se encuentra en el paquete "aulaserver".

La clase cuenta con un atributo: String url. El cual contiene
la dirección de la ruta de la base de datos.

Para que el programa pueda funcionar sin ningún problema, 
el desarrollador deberá cambiar el valor de esta variable
por uno nuevo que contenga la dirección actual de la base de datos.
